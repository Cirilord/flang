# Language

`flang` is a programming language project built in TypeScript.

## Summary

- Explicit static typing
- Clear and predictable syntax
- Spec-driven language evolution

## Documentation Structure

- `docs/language.md` is the high-level summary of the language
- `docs/variables.md` contains the current variable declaration and type annotation rules
- `docs/improvements.md` contains planned language improvements

At the current project stage, this split is enough.
More files should be added only when a topic grows enough to justify its own focused document.
